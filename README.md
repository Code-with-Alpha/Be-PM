# Be PM: Road to Prime Minister 🇮🇳

**Be PM: Road to Prime Minister** is an interactive, educational web game designed to teach the fundamentals of Indian election civics. Players embark on a journey starting as a "Common Citizen" and must navigate through 8 engaging levels of India's electoral process to ultimately become the Prime Minister. 

Guided by the witty and wise **Chunav Guru**, players will learn about voter registration, parliamentary structure, campaign rules, voting day procedures, and government formation in a fun, chat-based adventure!

---

## 🎮 Features

- **Interactive Chat Interface:** Talk to Chunav Guru, answer questions, and make decisions via a sleek, responsive chat UI.
- **8 Educational Levels:**
  - Level 1: The Voter Registration
  - Level 2: Understanding Parliament
  - Level 3: The Candidate Nomination
  - Level 4: The Campaign Trail
  - Level 5: Voting Day
  - Level 6: Counting Day
  - Level 7: Government Formation
  - Level 8: The Prime Minister
- **Rank Progression System:** Earn XP by answering correctly and rank up from "🌱 Common Citizen" all the way to "🏛️ PRIME MINISTER".
- **Dynamic Learning Tools:** Use built-in commands like `💡 Hint`, `🎯 Fun Fact`, and `📚 Explain More` to dive deeper into Indian civics.
- **Visuals & Polish:** Features a premium dark-mode glassmorphism design, floating particle backgrounds, and a confetti celebration system.
- **Certificate of Election:** Successfully complete the game to receive a custom-generated Prime Minister certificate!

## 🛠️ Technology Stack

This project is built using pure, vanilla web technologies. No external frameworks or libraries are required to run it.

- **HTML5:** App structure and layout.
- **CSS3:** Styling, animations, flexbox layouts, and custom properties (variables).
- **JavaScript (ES6):** Core game engine (`BePMGame`), state management, and UI logic.

## 🚀 How to Run

Running the game is incredibly simple as it is purely frontend-based.

1. Clone or download this repository.
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari).
4. Enter your name, select your age group (to tailor the learning experience), and begin your journey!

## 📂 Project Structure

- `index.html`: The main HTML shell containing the sidebar, chat interface, and canvas elements.
- `style.css`: All the styling rules, including animations, responsive design, and the UI's aesthetic.
- `game.js`: The heart of the application. It contains:
  - Game data (Questions, Options, Hints, Facts).
  - The `BePMGame` class which handles the game loop, user interactions, and visual canvas effects.
